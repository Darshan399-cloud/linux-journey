# Practice Tasks

## Task 1

Find all .txt files in the current directory.

```bash
find . -name "*.txt"
```

---

## Task 2

Find all directories.

```bash
find . -type d
```

---

## Task 3

Find files larger than 1 MB.

```bash
find . -size +1M
```

---

## Task 4

Find all empty files.

```bash
find . -empty
```

---

## Task 5

Find the location of the nano command.

```bash
which nano
```

---

## Task 6

Find the binary and man page of nano.

```bash
whereis nano
```
