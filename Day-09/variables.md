# Variables in Shell Scripting

## Definition

A variable is used to store data temporarily.

---

## Syntax

```bash
name="Darshan"
```

Display value

```bash
echo $name
```

---

## Example

```bash
city="Nandurbar"

echo "City: $city"
```

Output

```
City: Nandurbar
```

---

## Rules

- No space before or after =
- Variable names are case-sensitive
- Use meaningful variable names

Examples

```bash
age=22
course="Linux"
salary=50000
```
