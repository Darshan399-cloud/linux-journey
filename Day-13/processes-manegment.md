# Process Management

## What is a Process?

A process is a running instance of a program.

Example:

- Firefox
- Chrome
- VS Code
- Bash Terminal

## Important Commands

### Show Running Processes

```bash
ps
```

### Detailed Process List

```bash
ps -ef
```

### Live Process Monitor

```bash
top
```

### Kill Process

```bash
kill PID
```

### Force Kill

```bash
kill -9 PID
```
